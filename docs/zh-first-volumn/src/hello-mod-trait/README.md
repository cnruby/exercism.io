# 第三章 软件篋mod_trait_exerci

## 学习内容
- 学习软件篋文件、模块与程序结构
- 了解和学习动态调度关键词`dyn`
- 学习和理解动态与静态调度（Static vs Dynamic Dispatch）
- 衔接类型关键词`trait`作用
