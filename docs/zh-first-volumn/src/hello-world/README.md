# 关于软件篋项目hello-world

## 项目目标

　　学习和理解如何开发Rust语言的软件篋基本思路和方法，同时了解该项目的其他Cargo项目开发过程和方法。

## 主题内容

- 了解和学习Cargo工具作业区概念
- 学习和理解共享软件篋整个开发过程
- 了解和学习单元测试和集成测试基本概念
